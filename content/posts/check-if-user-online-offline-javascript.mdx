To check whether a user id online or offline in JavaScript, You have to listen for the events `online` and `offline` of the `window` in JavaScript.

## Check if User is online

Add an `online` event listener to the `window` like this,

```javascript
// to check if user is online
// add an online event listener
// to window
window.addEventListener("online", (event) => {

});
```
