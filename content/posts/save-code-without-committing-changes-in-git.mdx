To save the code changes on a branch without committing in Git, you can use the

```bash
git stash
```

command.

Now all the uncommitted code will be saved to the Git Stash and all the uncommitted code will be removed from the files too but that doesn't mean it's gone, it's saved ðŸ˜ƒ.

After doing this operation, you can perform any operation in Git including, changing branches, creating new commits, etc.

To get back the saved code from the stash, you can use the

```bash
git stash pop
```

command.

Now the uncommitted code which was in the stash is added again to the files.

Nifty Trick. Right ðŸŒŸ. I use this all the time when I have uncommitted code and want to save them but not commit it when switching between branches.
