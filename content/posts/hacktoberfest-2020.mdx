## What is Hacktoberfest ? ğŸŒŸ

HacktoberFest is an online fest conducted in October to support Open Source Contributions by DigitalOcean. It is a month-long contribution period where developers can register for the fest in the [Hacktoberfest](https://hacktoberfest.digitalocean.com/) website and make **4 Pull requests** in any open source projects in Github.

You have to make the 4 Pull requests between October 1 and October 31.

If you have managed to make 4 PR's then you can claim a supercool T-Shirt from DigitalOcean or plant a tree for your positive participation. Cool Right! ğŸ˜ƒ ğŸŒŸ

The Pull Requests doesn't need to be code related it can be adding some documentation, adding content, or designing a logo for a specific project, etc.
